# Portfolio_optimization Project
Code is return to optimize your investment portfolio and carry out simulation of different combination of stocks in your portfolio to & find out which
combination gives you best returns with less volatility.An Investment with high Sharpe Ratio and lowest volatality gives the best returns after risk adjusted 
returns
Libraries used in the Project
yfinance : To pull Stock data
pandas : To handle datastructures & data analysis
numpy : To handle multidimentional arrays
maplotlib : To handle Data visualizations
